Welcome to BPAN

BPAN is the Better Programming Alliance Network.

It is a a simple way to share library/module packages for any/all programming languages.

Currently it is in the proof-of-concept phase and being used for the Bash Programming language. See these links:
  • Bash BPAN
  • BPAN Packages List.
  • BPAN Commandline Client / Install Tool
To become a BPAN Contributing Author, simply visit the GitHub BPAN.org repository and Star it. Then you will show up on the BPAN Authors List.
