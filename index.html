BPAN: Bash Packages Ain't NoJoke™

A simple way to share library/module packages for any/all programming languages.

About

BPAN is currently in the proof-of-concept phase and being used for the Bash Programming language. See these links:

  • BPAN Documentation
  • BPAN Author List
  • BPAN Package List
  • BPAN Commandline Client / Install Tool

Becoming an author

To become a BPAN Contributing Author, simply visit the GitHub BPAN repository and Star it. Then you will show up on the BPAN Authors List.
