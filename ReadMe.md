bpan-client
===========

This is the command line tool for installing packages from BPAN.

## Synopsis

    > bpan install bashplus
    > bpan find json

## Installation

    > git clone http://github.com/bpan-org/bpan ~/.bpan
    > cd ~/.bpan
    > git checkout client
    > export PATH="$HOME/.bpan/bin:$PATH"
